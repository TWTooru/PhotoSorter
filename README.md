# Photo Sorter (照片挑片整理工具)

Photo Sorter 是一款使用 C# Windows Forms 開發的照片篩選工具。
為甚麼會想做這個？因為每當我跑完展完後照片不想一張一張整理，這東西就這樣誕生了。

---

## 核心特色

*   **RAW 檔自動連動**：篩選 JPG 照片時，自動比對並處理同名的 RAW 檔（支援 `.cr2`, `.nef`, `.arw`, `.dng` 等 30+ 種常見 RAW 格式）。
*   **影像檢視**：
    *   基於 `Matrix` 矩陣運算，支援以「滑鼠游標為中心」的無損縮放與平移拖曳。
    *   動態渲染畫質：拖曳時自動降低畫質確保流暢度，停止操作後瞬間恢復高畫質。
*   **復原機制**：支援無限制的堆疊復原（Undo），不小心刪錯照片只要按下 `Ctrl + Z` 即可立刻還原檔案與畫面狀態。
*   **EXIF自動轉向**：自動讀取照片 EXIF 方向屬性，正確顯示直式與橫式照片。
*   **深色/淺色主題**：內建 Dark Mode 與 Light Mode，減少長時間挑選照片的視覺疲勞（設定自動儲存於 `theme.cfg`）。

## 快捷鍵指南
| 快捷鍵 | 功能說明 |
| :--- | :--- |
| **Q** / **← 左方向鍵** | 上一張照片 |
| **E** / **→ 右方向鍵** | 下一張照片 |
| **R** | 順時針旋轉照片 90 度 |
| **Delete** | 刪除當前照片（將 JPG 與對應的 RAW 移至暫存區） |
| **Ctrl + Z** | 復原上一次的刪除動作 |
| **滑鼠滾輪** | 以游標為中心放大 / 縮小 |
| **滑鼠左鍵拖曳** | 平移放大後的照片 |
| **滑鼠左鍵雙擊** | 快速放大 200% / 恢復適應視窗大小 |

## 工作流程 (How to Use)
1.  **整理照片 (Organize)**
    *   點擊「整理照片」並選擇包含你所有照片的資料夾。
    *   軟體會自動建立一個 `raw` 子資料夾，並將所有 RAW 檔移入其中集中管理，畫面隨即載入 JPG 準備挑選。
2.  **篩選挑圖 (Culling)**
    *   使用快捷鍵瀏覽照片，遇到拍壞或不需要的照片按下 `Delete` 鍵。
    *   被刪除的照片會被移至自動建立的 `暫存區` 資料夾中（包含 JPG 與 RAW），不會立刻從硬碟永久抹除，保證安全。
3.  **執行轉移 (Transfer)**
    *   挑圖完成後，點擊「執行轉移」。
    *   系統會比對剩下的 JPG，將 `raw` 資料夾中**沒有對應 JPG** 的多餘 RAW 檔一併移至 `暫存區`。
    *   最後，你可以自行決定是否要將 `暫存區` 內的檔案丟進資源回收桶來釋放空間。

## 技術細節
*   **框架**：C# / .NET Windows Forms
*   **影像處理**：GDI+ (`System.Drawing`)
*   **效能優化**：Reflection 強制啟用雙重緩衝 (`DoubleBuffered`) 消除閃爍；使用 `System.Diagnostics.Stopwatch` 與 Timer 精準控制高/低畫質重繪時機。
